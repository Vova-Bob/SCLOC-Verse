# SCLOC-Verse

**Українська локалізація та голосовий асистент для Star Citizen.**

Десктопний застосунок для Windows, який спрощує встановлення, оновлення та
супровід української локалізації Star Citizen, а також голосового асистента
Л.І.А. Офіційний інструмент україномовної спільноти Star Citizen.

## Можливості

- **Українська локалізація** — встановлення, оновлення та видалення в один клік.
- **Усі середовища гри** — LIVE, PTU, HOTFIX, EPTU.
- **Автовизначення шляху Star Citizen** — або ручний вибір через провідник.
- **Голосовий асистент Л.І.А** — встановлення з GitHub, автоімпорт сертифіката,
  перевірка оновлень.
- **Hangar Timer** — таймер циклів ангара з накладанням (overlay) поверх гри.
- **Очищення кешу гри** — видалення shader-кешу тощо без втрати прогресу.
- **Гарячі клавіші** — налаштовувані хоткеї для швидкого доступу.
- **Автооновлення** — застосунок сам перевіряє та встановлює нові версії.
- **Обліковий запис Discord** — вхід через Discord OAuth.

## Вимоги

- **Windows 10/11 (x64)**.
- Встановлена гра **Star Citizen** (LIVE/PTU/HOTFIX/EPTU).
- ~150 МБ вільного місця.

## Встановлення

1. Завантажте останній `SCLOC-Verse_Setup.exe` зі сторінки
   [Releases](https://github.com/Vova-Bob/SCLOC-Verse/releases).
2. Запустіть інсталятор (може знадобитись дозвіл SmartScreen — застосунок
   не має платного сертифіката).
3. Запустіть SCLOC-Verse з меню «Пуск» або з робочого столу.
4. На вкладці «Налаштування» вкажіть шлях до папки Star Citizen
   (автопошук або вручну).
5. На вкладці «Локалізація» оберіть середовище гри та натисніть «Встановити».

> Перевірте цілісність завантаженого файлу за допомогою `.sha256`,
> який додається до кожного релізу.

## Використання

| Вкладка | Призначення |
|---|---|
| **Головна** | Швидкий доступ до всіх функцій та картки спільноти. |
| **Локалізація** | Встановлення / оновлення / видалення української локалізації. |
| **Л.І.А** | Встановлення голосового асистента та його оновлення. |
| **Hangar** | Таймер циклів ангара з overlay-вікном. |
| **Налаштування** | Шлях до гри, кеш, гарячі клавіші, обліковий запис. |

Після встановлення або оновлення локалізації **перезапустіть гру**, щоб
зміни застосувались.

## Збірка з сирців

Потрібні:

- [.NET 9 SDK](https://dotnet.microsoft.com/download) (або новіший).
- [Inno Setup 6](https://jrsoftware.org/isinfo.php) (для інсталятора).
- Git.

```powershell
git clone https://github.com/Vova-Bob/SCLOC-Verse.git
cd SCLOC-Verse

# Release-збірка + інсталятор (автоматизовано)
cd Installer
.\build-installer.ps1
```

Готовий інсталятор з'явиться у `Installer/Output/SCLOC-Verse_Setup.exe`
разом із `.sha256` контрольною сумою. Скрипт також перевіряє відсутність
mojibake у сирцях перед збіркою.

## Технології

- **WPF + .NET 9** (C#).
- **Supabase** — автентифікація (Discord OAuth) та метадані інсталяцій.
- **Inno Setup 6** — інсталятор.
- **Single-file self-contained publish** — один `.exe` без залежностей.

## Приватність

- Аутентифікація — виключно через Discord OAuth (scope `identify`).
- Зберігаються лише метадані інсталяції (версія застосунку, платформа, країна
  через GeoIP, час останнього запуску).
- Поведінкова телеметрія **не збирається**.
- Усі таблиці захищені Row Level Security (RLS).
